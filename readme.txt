== Description ==
A polyfill for the Adobe's CSS3 proposal for Text regions (Tested + Works on Chrome, Safari, Firefox, IE 8+)
For more information visit [GitHub](https://github.com/adobe-webplatform/css-regions-polyfill)

From 1 October 2016 on, this plugin is no longer supported. Please use [Powerpack](http://www.ramoonus.nl/wordpress/powerpack) instead.

== Installation ==
1. Upload `regionsjs/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
